# CONNECT_NUTRI

O ConnectNutri é uma plataforma web desenvolvida com o objetivo de otimizar o gerenciamento de consultas nutricionais, promovendo maior praticidade, organização e acessibilidade tanto para pacientes quanto para profissionais da área da nutrição que atuam na clínica. O sistema foi projetado com foco na experiência do usuário (UX) e na construção de uma interface intuitiva e adaptável, permitindo que os usuários realizem operações relacionadas ao agendamento de consultas de forma simples, eficiente e segura.

# OBJETIVO DO POJETO E FUNCIONALIDADES

O projeto tem como principal objetivo desenvolver uma solução digital capaz de automatizar e facilitar o processo de gerenciamento de consultas nutricionais.

A plataforma foi idealizada para oferecer uma experiência moderna e funcional, permitindo:

- Realização de agendamentos online;
- Remarcação de consultas de forma dinâmica;
- Cancelamento de consultas com justificativa;
- Organização automatizada da agenda do nutricionista;
- Redução de conflitos de horários e falhas operacionais;
- Centralização das informações relacionadas aos atendimentos.

O sistema também visa aplicar conceitos de desenvolvimento web, banco de dados e design de interfaces centradas no usuário.

# ARQUITETURA DO SISTEMA

O sistema foi estruturado utilizando conceitos básicos de desenvolvimento web e integração com banco de dados relacional, permitindo maior organização das informações e melhor gerenciamento das funcionalidades da aplicação. O foco o desenvolvimento visa garantir:

- Integridade dos dados;
- Organização das informações dos pacientes;
- Controle de agendamentos;
- Armazenamento eficiente das consultas realizadas.
